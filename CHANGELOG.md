# Changelog

Toutes les modifications notables de ce projet seront documentées dans ce fichier.

Le format est basé sur [Keep a Changelog](https://keepachangelog.com/fr/1.0.0/),
et ce projet adhère au [Semantic Versioning](https://semver.org/lang/fr/).

## [0.1.0] - 2025-11-06

### Ajouté
- Composant `Button` avec 4 variants (primary, secondary, outline, ghost)
- Composant `Input` avec label, message d'aide et gestion d'erreurs
- Support des tailles (sm, md, lg) pour tous les composants
- Support du dark mode via les variables CSS Blick
- Import automatique de la feuille de style Blick (https://utils.blick.ch/static/global/css/base.css)
- Configuration Storybook pour le développement et la documentation
- Build avec Vite pour ESM et CJS
- Définitions TypeScript complètes
- Classes CSS utilitaires personnalisées

### Documentation
- README.md avec guide d'installation et utilisation
- USAGE.md avec exemples détaillés
- CONTRIBUTING.md avec guide de contribution
- Stories Storybook pour chaque composant
